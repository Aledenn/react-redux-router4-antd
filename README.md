#### 项目：招聘网站
#### 实现功能：
  - 注册登录
  - 权限划分
  - 聊天
  - 个人信息
  - 聊天emoji
#### 使用技术：
  前端：react+redux+router4+antdmobile
  后端：nodejs+express+mongodb
#### 使用：
mac下安装mongodb和启动mongodb
```
brew install mongodb 
brew services start mongodb
```
运行项目
```
//安装依赖
npm install 

//运行前端
npm run start

//运行后端
npm run server
```
